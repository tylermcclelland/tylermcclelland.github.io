A step-by-step guide to building and deploying Value AI on your personal website. This guide covers repository setup, development environment, frontend and backend implementation, and deployment strategies.

1. Project Structure & Repository Setup

  1. Initialize a GitHub repository:
    • Create a new repo named value-ai on GitHub.
    • Clone locally:
      git clone https://github.com/yourusername/value-ai.git
      cd value-ai
  2. Add .gitignore: 
    # Node
        node_modules/
        /# Python
        .venv/
        __pycache__/
        *.pyc
        # Env files
        .env
  3. Create main branches: git checkout -b main
    git checkout -b develop
    git push -u origin main develop

2. Development Environment

2.1 Backend (FastAPI + Uvicorn)

  1. Set up Python virtual environment: python3 -m venv .venv
    source .venv/bin/activate
  2. Install dependencies: pip install fastapi uvicorn python-dotenv openai
  3. Create backend/main.py: 
    import os, base64
        from dotenv import load_dotenv
        from fastapi import FastAPI, File, UploadFile
        from fastapi.middleware.cors import CORSMiddleware
        from openai import OpenAI
        
        load_dotenv()
        client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))
        app = FastAPI()
        app.add_middleware(CORSMiddleware, allow_origins=["http://localhost:5173"], allow_methods=["POST"], allow_headers=["*"])
        
        @app.post("/upload")
        async def upload_image(image: UploadFile = File(...)):
            img_bytes = await image.read()
            b64 = base64.b64encode(img_bytes).decode()
            data_url = f"data:{image.content_type};base64,{b64}"
            response = client.responses.create(
                model="gpt-4o",
                input=[{...}]
            )
            return {"value": response.output_text}
  4. Run server: uvicorn backend.main:app --reload --port 8000

2.2 Frontend (React + Vite)

  1. Install Node.js & npm: brew install node
  2. Scaffold Vite + React: npm create vite@latest frontend -- --template react
    cd frontend
    npm install
  3. Add environment file frontend/.env: 
    VITE_API_URL=http://localhost:8000
  4. Implement App.jsx & CSS: Copy the sample React component and CSS from the guide.
  5. Run frontend: npm run dev

3. Testing Locally

  • Backend: curl -X POST http://localhost:8000/upload -F 'image=@/path/to/car.jpg'
  • Frontend: Open http://localhost:5173, upload an image, and click Get Value.
